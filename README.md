## Installation

1. Clone this repository:

```bash
git clone https://github.com/dreambabyyy/Dropee.git
cd Dropee
```

2. Install dependencies:

```bash
npm install
```

### Setting Up Accounts
```bash
nano data.txt
```
```bash
nano proxy.txt
```
1. Edit `data.txt` file in the root directory
2. Add accessToken (one per line) for each account

```
user_id
query_id
```
